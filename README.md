 Spor Salonu Yönetim Sistemi 

Bu proje, Web Programlama dersi kapsamında ASP.NET Core MVC kullanılarak geliştirilmiş bir Spor Salonu Yönetim ve Randevu Sistemidir.

 Projenin Amacı
Spor salonu üyelerinin kayıtlarını tutmak, antrenörlerin uzmanlık alanlarını yönetmek ve üyelerin antrenörlerden randevu almasını sağlamaktır. Ayrıca sistem, üyelerin vücut kitle indeksine (VKİ) göre yapay zeka tabanlı egzersiz önerileri sunmayı hedefler.

Üye Yönetimi: Yeni üye kaydı, fiziksel ölçümler (Boy, Kilo) ve sağlık bilgileri takibi.
Antrenör Yönetimi: Eğitmenlerin uzmanlık alanları ve çalışma saatlerinin tanımlanması.
Randevu Sistemi: Üyelerin müsait antrenörlerden ders/randevu alabilmesi.
Akıllı Öneri Sistemi: Üyenin fiziksel verilerine göre (Kardiyo veya Ağırlık) antrenman önerisi.


 Kullanılan Teknolojiler
* .NET 8.0
* ASP.NET Core MVC
* Entity Framework Core (Veritabanı)
* Bootstrap 5 (Tasarım)
* Identity (Üyelik ve Yetkilendirme)
